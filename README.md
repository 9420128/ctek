#  Развёртывание проекта

Проект демонстрирует работу Vue 3 с Webpack и исправленную логику `$attrs`/`mergeProps`.

---

## ️ Установка проекта

```bash
git clone https://github.com/9420128/ctek.git
cd ctek
npm install
npm run dev
```
Патч автоматически применяется после установки зависимостей.

Применить вручную:
```bash
npm run postinstall
```
